# Wave 1 learnings <!-- oc:id=sec_aa -->

- Renderer has no full reconciliation pass after SSE reconnect or server switch; `startEventLoop()` only reopens stream.
- Renderer flushes streaming buffers on `session.status { type: "idle" }` only; `session.idle` currently bypasses flush path.
- Electron local presence is attach-only in `apps/desktop/src/main/opencode-manager.ts`; browser/server route already supports inferred local sessions.
- Sidebar, command palette, and tray each compute active/recent differently.
- Canonical design doc now lives in `docs/session-sync-reconciliation.md`; evidence for tasks 1-4 seeded under `.sisyphus/evidence/session-sync/`.
